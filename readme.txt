Software Requirements:
- Arduino IDE
- Processing IDE
- Adafruit AMG88XX Library
- Bluetooth graphics android App

Hardware Requirements:
- Arduino Nano
- Adafruit AMG88XX Grid Eye sensor module
- HC-05 or HC-06 Bluetooth module

Connections:

Grid Eye - Arduino Nano

Vcc - 3.3v
Gnd - Gnd
SDA - A4
SCL - A5

Arduino Nano - HC-05

Vcc - Vin
Gnd - Gnd
TX - RX
RX - TX
